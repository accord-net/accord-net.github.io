Remarks

              This class helps sort the elements of an array without swapping
              elements which are already in order. This comprises a stable
              sorting algorithm. This class is used by the StableSort T ( T , Comparison T )
              method to produce a stable sort of its given arguments.
            
Examples

              In order to use this class, please use StableSort T ( T , Comparison T ).
