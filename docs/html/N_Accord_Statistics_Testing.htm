Accord.Statistics.Testing Namespace Accord.NET Framework
Show table of contents (goes to the online documentation index).
Contains 15+ statistical hypothesis tests, including ANOVA tests, non-parametric tests such as the Kolmogorov-Smirnov test, contingency table tests such as the Kappa test, the Bhapkar and Bowker tests and the more common Chi-Square, Z, F, T and Wald tests.
