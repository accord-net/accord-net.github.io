Returns a point, which is the furthest away from the specified line.
Remarks

The method finds the furthest point from the specified line. Unlike the GetFurthestPointsFromLine(IEnumerableIntPoint, IntPoint, IntPoint, IntPoint, IntPoint) method, this method find only one point, which is the furthest away from the line regardless of side from the line.
