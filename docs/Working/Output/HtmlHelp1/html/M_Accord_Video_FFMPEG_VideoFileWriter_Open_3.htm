Remarks

The methods creates new video file with the specified name. If a file with such name already exists in the file system, it will be overwritten.

When adding new video frames using WriteVideoFrame(Bitmap) method, the video frame must have width and height as specified during file opening.

 Note
The bit rate parameter represents a trade-off value between video quality and video file size. Higher bit rate value increase video quality and result in larger file size. Smaller values result in opposite – worse quality and small video files.
